ChatPaste for macOS

ChatPaste is a local macOS menu bar utility that converts copied Markdown, plain text, browser rich text, or RTF into an Outlook-friendly clipboard payload.

  1. Copy Markdown, plain text, browser rich text, or RTF content.
  2. Choose ChatPaste from the macOS menu bar.
  3. Select Convert Clipboard for Outlook, or press Control + Option + Command + V.
  4. Paste manually into Outlook for Mac or another rich text destination.
  • If nothing converts, confirm the clipboard contains selectable text, HTML, or RTF.
  • If formatting looks different in Outlook, try simpler spacing and table border settings.
  • If the hotkey does not respond, use the menu bar command and confirm ChatPaste is running.
  • If RTF is unavailable, paste into Outlook anyway; HTML and plain text are still copied.
  • If you want to undo a conversion, choose Restore Previous Clipboard from the menu bar.
